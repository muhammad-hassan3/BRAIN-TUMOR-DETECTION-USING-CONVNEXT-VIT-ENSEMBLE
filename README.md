# Brain Tumor Detection Using ConvNeXt + Vision Transformer Ensemble

## Overview

This project uses a hybrid deep learning architecture combining ConvNeXt and Vision Transformer (ViT) models to classify brain MRI scans into four categories:

* Glioma
* Meningioma
* Pituitary Tumor
* No Tumor

The project includes a Streamlit web application for real-time MRI image analysis and Grad-CAM visualization.

## Features

* MRI Brain Tumor Classification
* ConvNeXt + ViT Ensemble Model
* Streamlit Web Application
* Grad-CAM Visualization
* PyTorch Implementation

## Technologies Used

* Python
* PyTorch
* Streamlit
* NumPy
* Matplotlib
* TIMM

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
